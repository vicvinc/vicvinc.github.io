# Hello, World

Be patient. Be modest.

这是我的第一篇博客。没什么技术内容，只是一个开始。

写博客这件事想了很久，但一直觉得"还没准备好"——技术不够牛、文笔不够好、内容不够深。后来想通了：**如果等到准备好了才开始，那可能永远不会开始。**

所以这里不追求篇篇精华，也不追求教程级别的严谨。只记录学习中真正的所思所得，踩过的坑、想明白的事、写出来的代码。

> Waiting for the sunrise, waiting for the sunshine.

这是 2017 年的开篇。希望多年后回头，能看到一条虽然曲折但一直在延伸的路。
